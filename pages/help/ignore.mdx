# Ignore

Syntax: ignore \<player\>  
Syntax: ignore self  

Allows you to IGNORE any tells, replies, beeps etc. from a certain player.  
Only one player can be ignored at any one time by you.  

IGNORE SELF, will cancel the ignore.
