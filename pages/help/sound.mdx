# Sound

Dark Wizardry used to have a soundpack in the late 90s, but it has disappeared.  
For now we don't use sound TOO much, but hopefully in the near future this will  
change soon.  

To turn sound OFF just type config -sound.  

If you would LIKE the sound messages, type config +sound
