# Shop

---  
|                                                                   |  
| There are several helpful commands when in shops. LIST shows you  |  
| the items in the shop, BUY \<item\> to buy it, SELL \<item\> to       |  
| attempt to sell an item, VALUE \<item\> to see what the shopkeeper  |  
| will give you for your item, and BROWSE \<item\> to find out more   |  
| about an item that is dor sale. See HELP \<command\> for more.      |  
|                                                                   |
