# Browse

---  
| Syntax: BROWSE \<amount|all\>                                         |  
|                                                                   |  
| The browse command functions as a free way to identify shop items |  
| before they are purchased.  To use it, simply type: browse \<item\> |  
| For example, 'browse sword' or 'browse 3.sword'.                  |
