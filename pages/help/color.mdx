# Color

Syntax: color \<defaults|list|set|toggle\> \<field #\> \<color\>  

'defaults' will set your colors back in default.  'list' will show colors  
currently set.  'toggle' wil toggle your colors ON or OFF.  

With the 'set' option you can set your own color preferences.  
It will work like this: color set \<#field\> \<color\>.  
\<color\> must be one of:  
    r R : red / light red  
    g G : green / light green  
    y Y : yellow / light yellow  
    b B : blue / light blue  
    m M : magenta / light magenta  
    c C : cyan / light cyan  
    w W : grey / white  
    ?   : random  
    d   : dark
