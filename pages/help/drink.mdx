# Drink

Syntax: drink \<object\>  
Syntax: eat   \<object\>  
Syntax: fill  \<object\>  
Syntax: empty \<object\>  

When you are thirsty, DRINK something.  

When you are hungry, EAT something.  

FILL fills a drink container with liquid from a fountain.  

EMPTY empties a drink container so it can be refilled again.
