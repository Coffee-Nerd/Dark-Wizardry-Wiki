# Notify

Syntax: notify \<+/- channel\>  

NOTIFY gives you additional information about certain events in the  
game. If such an event is triggered notify will send you a message.  
For example notify can send you a message if a player enters the realm.  

To see what channels are available with notify type "notify". To toggle  
a notify channel on/off, type notify +channel or notify -channel.
