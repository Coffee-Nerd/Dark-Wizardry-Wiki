# DarkMUSH Client

This section contains documentation for the DarkMUSH client features and windows, designed specifically for Dark Wizardry MUD.

## Available Windows

- [Affects Window](/DarkMUSH/affectswindow) - Displays active effects on your character
- [ASCII Map Window](/DarkMUSH/asciimapwindow) - Shows a visual map of your surroundings
- [Group Window](/DarkMUSH/groupwindow) - Displays information about your group members
