# Races in Dark Wizardry

Dark Wizardry supports a wide variety of races. Their core attributes are defined in three groups:
- **Initial Attributes**: The starting values.
- **Natural Attributes**: The innate or “with equipment” values.
- **Max Attributes**: The highest attainable values.

Below the races are organized by their remort category. In each race entry, the ability flags are rendered as clickable inline links so you can quickly access the corresponding help page.

## How to Use This Documentation

In this documentation each race’s entry shows:
- Its name with the short name in parentheses and key abilities.
- A table summarizing the three sets of core attributes:
  - **Initial Attributes**: The base starting values.
  - **Natural Attributes**: The inherent values (often reflecting the racial bonus when equipped).
  - **Max Attributes**: The highest attainable values after training and equipment.

Happy adventuring in Dark Wizardry!
